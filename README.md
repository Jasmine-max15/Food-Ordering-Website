🍔 Food Ordering Website

A simple and responsive **Food Ordering Web Application** built using **HTML, CSS, and JavaScript**.
This project allows users to browse food categories, view items, and place orders through a clean UI.

🚀 Features

* 🏠 Home page with food highlights
* 🍕 Food categories (Pizza, Burger, Sandwich, etc.)
* 🔍 Search functionality
* 📋 Food listing with images and prices
* 🛒 Order page
* 🔐 Login page UI
* 📞 Contact page

 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome (for icons)
